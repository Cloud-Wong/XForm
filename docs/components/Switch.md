## Props

| 名称       | 类型     | 默认值  | 描述       |
| ---------- | -------- | ------- | ---------- |
| label      | `string` | ''      | 组件标题   |
| onText     | `string` | ''      | 打开时文案 |
| offText    | `string` | ''      | 关闭时文案 |
| default    | `bool`   | `false` | 默认值     |
| helperText | `string` | ''      | 提示       |
