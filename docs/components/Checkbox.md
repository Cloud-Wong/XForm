# Checkbox

多选框，允许在多个选项中选中多个状态。

<code src="./examples/Checkbox" compact background="#fff"></code>


## Props

| 名称       | 类型     | 默认值  | 描述     |
| ---------- | -------- | ------- | -------- |
| label      | `string` | ''      | 组件标题 |
| default    | `array`  | []      | 默认值   |
| options    | `array`  | []      | 选项配置 |
| row        | `bool`   | `false` | 水平布局 |
| helperText | `string` | ''      | 提示     |
