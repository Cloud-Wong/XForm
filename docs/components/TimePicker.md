# TimePicke

时间选择器，用于输入或选择时间。

<code src="./examples/TimePicker" compact background="#fff"></code>


## Props

| 名称       | 类型     | 默认值  | 描述               |
| ---------- | -------- | ------- | ------------------ |
| label      | `string` | ''      | 组件标题           |
| default    | `array`  | []      | 默认值             |
| options    | `array`  | []      | 选项配置           |
| helperText | `string` | ''      | 提示               |
| checkbox   | `bool`   | `false` | 选项卡为复选框样式 |
| chip       | `bool`   | `false` | 值显示为标签样式   |
