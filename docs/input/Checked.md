# Checked

单个选项框，返回 `true` 或者 `false`。


## 示例

<code src="./examples/Checked" compact background="#fff"></code>


## Props

| 名称       | 类型     | 默认值 | 描述                               |
| ---------- | -------- | ------ | ---------------------------------- |
| label      | `string` | ''     | 组件标签                           |
| title      | `string` | ''     | 选项标题，支持 Markdown 格式的链接 |
| default    | `array`  | false  | 默认值                             |
| helperText | `string` | ''     | 提示                               |
