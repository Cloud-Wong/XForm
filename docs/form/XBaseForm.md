# XBaseForm

| 名称   | 类型     | 默认值  | 描述                         |
| ------ | -------- | ------- | ---------------------------- |
| path   | `string` | `-`     | 状态管理对象中表单数据的路径 |
| store  | `Mobx`   | `-`     | 存储数据的状态管理对象       |
| schema | `json`   | `{}`    | 配置协议                     |
| debug  | `bool`   | `false` | 是否展示调试面板             |
